From the Ground Up

Introduction

The purpose of this page is to demonstrate the concepts of building up a website from the ground up, starting with an empty HTML document.

Building Blocks

HTML consists of elements that, when combined, make up the architecture of a website. Common elements include:

  • Headings
  • Paragraphs
  • Lists
  • Links
  • Images
  • Tables

Headings

Headings are used to describe the content of a page or section and should be used in a hierarchical manner. The most important heading on a page should be an h1 element, and subsequent headings should use h2, h3, h4, h5, and h6 elements to create a heading hierarchy.

Paragraphs

Paragraphs are used to contain chunks of text that together form a readable section of content. Long paragraphs can be broken up into smaller sections using the p element.

Lists

Lists are used to present related information in a logical and organized manner. There are two types of lists: unordered and ordered. Unordered lists use the ul element and list items are marked up with the li element. Ordered lists use the ol element and list items are marked up with the li element.

Links

Links are used to navigate between pages on a website. Links can also be used to point to external websites. Links are marked up with the a element and the href attribute.

Images

Images are used to illustrate or supplement a piece of content. Images are marked up with the img element and the src attribute.

Tables

Tables are used to present data in a tabular format. Tables are marked up with the table element and rows and columns are marked up with the tr and td elements respectively.

Conclusion

This page has demonstrated the basic building blocks of an HTML document and how they can be used to create a functioning website.
